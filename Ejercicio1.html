Definición

HTML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado HTML) para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros…



Aplicaciones:

Una aplicación HTML5, por otro lado, se refiere a una aplicación móvil construida completamente con HTML, CSS y Javascript solamente. Las aplicaciones HTML5 son aplicaciones web y deben ejecutarse utilizando el navegador del sistema operativo subyacente.



Versiones:

HTML 5, HTML 5.1, HTML 5.2 HTML 5

Incorpora nuevos elementos no contemplandos en HTML 4.01. Hay diversos cambios respecto a HTML 4.01. Hay nuevas etiquetas, se introduce la posibilidad de introducir audio y video de forma directa en la web sin necesidad de plugins o complementos en los navegadores, y otras novedades.
